#### Backend:
- **Java EE** microservice <3 **Java 9**

  - backed by **PostgreSQL**

  - connected to
    - **Akka**
    - **OpenZipkin** tracing
    - **Kafka** messaging
    - **Prometheus**
  - deployed to **Payara 5**
- **Spring Boot** microservice <3 **Kotlin**

  - backed by **CockroachDB**

    - **Akka**
    - **OpenZipkin** tracing
    - **Kafka** messaging
    - **Prometheus**
- **Play 2** microservice <3 **Scala**
  - running on Play
  - backed by **Cassandra**
  - connected to
    - **Akka**
    - **OpenZipkin** tracing
    - **Kafka** messaging
    - **Prometheus**

    - **Kafka** messaging
    - **Prometheus**
